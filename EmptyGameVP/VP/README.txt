An Empty game solution with settings for Visual Pinball.

Includes settings for the service mode that can adjust the tables configuration without a player having to edit scripts.

- Table Physics
- Flipper Physics

- Translite (DirectB2S) = Saves user having to edit script they can enable this inside the menu.

- Table sounds = Players often use cabinets with real solenoids so this is usefull to 	disable sounds.
